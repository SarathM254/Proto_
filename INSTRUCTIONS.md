## If You Get Network Errors

1. **Make sure the server is running**:
   - Look for the message: "Proto Backend Server running on http://localhost:3000"
   - If not running, double-click `run-server.bat`

2. **Check the URL**:
   - Always use: `http://localhost:3000`
   - Don't open the HTML files directly

3. **Clear browser cache**:
   - Press Ctrl+F5 to refresh

## Troubleshooting

### Server Not Starting?
1. Make sure you're in the Proto folder
2. Double-click `run-server.bat`
3. Wait for "Server running" message

### Still Getting Network Errors?
1. Check if port 3000 is free
2. Try closing other applications
3. Restart the server

### Profile Not Working?
1. Make sure you're logged in
2. Click the profile icon in bottom navigation
3. Try logging out and back in
